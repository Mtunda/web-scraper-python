## Description
This project scrapes quotes and authors from a website and saves the data into a CSV file.

## Tools Used
- Python
- BeautifulSoup
- Requests
- Pandas

## Features
- Scrapes multiple pages
- Extracts quotes and authors
- Saves data into CSV format

## Output
A CSV file containing scraped quotes and their authors.

## How to Run
1. Install dependencies:
   pip install requests beautifulsoup4 pandas

2. Run the script:
   python scraper.py
